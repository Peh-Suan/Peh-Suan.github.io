pymi

Description

  • Huang Po-Hsuan

  • View on Github

  • Last updated 9 Nov., 2023

pymi allows you to: 1)calculate the MI (mutual information) and AMI (average mutual information) between each pair of bigram in a given corpus, 2) plot sentences into trees based on the MI/AMI, 3) segment sentences based on the MI/AMI, and 4) plot the mean number of word distribution for different thresholds of MI/AMI.
