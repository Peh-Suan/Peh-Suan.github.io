Ultrasonic-and-optical-imaging-assisted speech correctness judgment model

Description

  • Huang Po-Hsuan

Ultrasonic-and-optical-imaging-assisted speech correctness judgment model is an automated judgment model that assesses the correctness of speech production for second language acquisition/speech therapy purposes with the combination of both ultrasnic and optical images. This is an on-going project yet to be put onto Github. Feel free to contact me for the source codes.
