Ultrasonic-and-optical-imaging-assisted speech correctness judgment model

Description

Ultrasonic-and-optical-imaging-assisted speech correctness judgment model is an automated judgment model that assesses the correctness of speech production for second language acquisition/speech therapy purposes with the combination of both ultrasnic and optical images.
